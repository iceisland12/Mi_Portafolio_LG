## 🚀 Deploy

Simplemente sube los archivos a cualquier hosting estático:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

## 📝 Personalización

1. Reemplaza `assets/images/profile.jpg` con tu foto
2. Agrega tu CV PDF en `assets/documents/`
3. Actualiza los enlaces sociales en el footer
4. Modifica colores en `:root` en `style.css`
